## Submission Process
1. https://developer.crazygames.com/games
2. Can't seem to modify a rejected game, so create a new submission.
3. Images need to be captured for:
   1. Landscape 16:9 (1920x1080)
   2. Portrait 2:3 (800x1200)
   3. Square 1:1 (800x800)
4. Preview video Landscape is 852x480 and less than 20 seconds
